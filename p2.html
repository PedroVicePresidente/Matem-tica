Pergunta 2

Mostre onde a sequência de Fibonacci aparece na natureza.

A sequência de Fibonacci é uma sequência numérica definida recursivamente. Os termos são: F0 = 0, F1 = 1
Para n ≥ 2: Fn = Fn−1 + Fn−2

Ou seja, cada termo (a partir do segundo) é a soma dos dois termos anteriores.

Exemplo: termos até a 15ª casa
